# seamless-dask

`seamless-dask` provides Dask-based distributed execution support for Seamless:

- scheduler and worker integration helpers
- Seamless-aware Dask client components
- command-line wrapper for launching workers with Seamless context

## Installation

```bash
pip install seamless-dask
```

## CLI scripts

Installing `seamless-dask` also provides:
